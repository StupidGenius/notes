Markdown

Markdown is a markup langauge which is used when writing notes on this site. Below is a quick start guide on the language to get you started.

Headers

# Header #

Header

## Sub header ##

Subheader

### Subsubheader ###

Subsubheader

Links

[Link](http://google.com)

Link

Blockqoute

> Quoted text

Quoted text

Code

Indenting the text with at least 4 spaces to put the text in a code block.

    Some Code

Some Code

A code block can also be made in-line with backticks

`Some Code`

Some Code

Lists

Unordered lists can be made using +,- or *

* Item one
* Item two
* Item three
  • Item one
  • Item two
  • Item three

Ordered lists can be made by typing a number and then a period.

1. Item one
2. Item two
  1. Item one
  2. Item two

Horizontal lines

A horizontal line can be made by using 3 or more dashes.

-----

Emphasis

Adding one underscore or star (*) as prefix and suffix on a word formats the word in italic.

_italic_ *text*

italic text

Using two underscores or stars, formats the word in bold.

__bold__ **text**

bold text

Escaping

If you want to use for example two stars before and after a word without making it bold, the characters needs to be esaped by adding a backslash infront of hte characters.

\_\_This\_\_ word is not in bold

__This__ word is not in bold
